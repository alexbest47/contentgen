# ContentGen - Сгенерированный контент для 5 программ

## Обзор

Этот каталог содержит полный набор материалов для 5 профессиональных программ:

1. **Профессия Гештальт-терапевт**
2. **Профессия Декоратор интерьера**
3. **Профессия Дизайнер Одежды**
4. **Профессия Интегративный нутрициолог**
5. **Профессия КПТ-терапевт**

## Структура файлов

Для каждой программы создано 3 файла:

### 1. test_diagnostic.txt
**Содержание**: Диагностический тест с 12-15 вопросами

**Структура**:
- Название теста (кликабейт-формат, основанный на боли/желании аудитории)
- Описание теста (1-2 предложения)
- 12-15 вопросов различных типов:
  - `single-choice` (с autoAdvance: true) - 50-70%
  - `multi-choice` (с maxSelections) - 20-30%
  - `scale` - 5-10%
  - `game-psychology` (emotion-wheel, figure-placement, body-map) - опционально
  - `color-choice` - опционально
- Логика результатов (3-5 типов профилей)

**Специфические вопросы**:
- Вопрос 1: "Сколько вам лет?" (обязательный, autoAdvance: true)
- Второй с конца: Вопрос о предыдущем обучении
- Последний: "С чем связан ваш интерес?" с вариантами:
  - Выбираю новую профессию
  - Хочу усилить текущую деятельность
  - Интересно для себя

**Примеры результатов**:
- Профайл пользователя
- Описание
- Ключевые маркеры
- Инсайт
- Мост к программе (как перейти к обучению)

### 2. pdf_lead_magnet.txt
**Содержание**: Описание PDF-лид-магнита для каждой программы

**Структура**:
- PDF-ЛИД-МАГНИТ + название программы
- Название материала (кликабейт с конкретным обещанием)
- Тип материала (одно из):
  - Рабочая тетрадь
  - Чек-лист
  - Инструкция
  - Гид
  - Справочный материал
  - Экспертный материал
- Описание (1-3 предложения о проблеме и решении)
- Содержание материала
- Целевая аудитория
- Решаемая проблема
- Уникальность
- Почему скачивать

### 3. diagnostic_image.png
**Содержание**: Сгенерированное изображение для визуализации программы

**Характеристики**:
- Формат: PNG с прозрачным фоном
- Размер: Квадратный (1:1 соотношение)
- Стиль: Современный, мечтательный, цифровой
- Примерный размер файла: 600KB - 2MB
- Сгенерировано через OpenRouter API (Google Gemini 3.1 Flash Image Preview)

## Описание контента по программам

### 1. Профессия Гештальт-терапевт
- **Целевая аудитория**: Люди 28-55 лет в кризисе смены профессии
- **Тест**: "Готов ли я стать гештальт-терапевт?"
- **Лид-магнит**: "7 способов вернуться в контакт с собой прямо сейчас"
- **Образ**: Спокойный, интроспективный, с геометрическими формами (мысли/эмоции)

### 2. Профессия Декоратор интерьера
- **Целевая аудитория**: Женщины 30-49 лет, мечтающие о творчестве
- **Тест**: "Готова ли я к карьере Декоратора интерьера?"
- **Лид-магнит**: "Цветовая гармония: Быстрый старт для декоратора"
- **Образ**: Эстетичное, красивое пространство с цветовыми переходами

### 3. Профессия Дизайнер Одежды
- **Целевая аудитория**: Творческие люди, поклонники моды
- **Тест**: "Готов(а) ли я стать дизайнером одежды?"
- **Лид-магнит**: "Как создать силуэт: 5 основных приемов дизайнера"
- **Образ**: Модные эскизы, ткани, элегантные силуэты

### 4. Профессия Интегративный нутрициолог
- **Целевая аудитория**: Люди, интересующиеся здоровьем и питанием
- **Тест**: "Готов(а) ли я стать интегративным нутрициологом?"
- **Лид-магнит**: "5 принципов интегративного питания"
- **Образ**: Свежие продукты, травы, баланс восточных и западных символов

### 5. Профессия КПТ-терапевт
- **Целевая аудитория**: Опытные психологи, ищущие специализации
- **Тест**: "Готов(а) ли я стать КПТ-терапевтом?"
- **Лид-магнит**: "КПТ-техники для работы с тревогой"
- **Образ**: Профессиональное, научное, с визуализацией мысли и структур

Все файлы закодированы в UTF-8 и готовы к интеграции в CMS.
