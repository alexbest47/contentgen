### Как использовать шаблон для нового курса

#### 1. Скопируйте папку
Сделайте копию всей папки `talentsy-template/` и переименуйте.

| **Для кого** (`#whom`) | Заголовок, описание, изображение |
| **25% студентов** (`.s2-hg8ap7v1d`) | Статистика рекомендаций |

| **10 подходов** (`.s24-hg8ap7v1d`) | Список подходов |
| **Практика** (`.s25-hg8ap7v1d`) | Виды практик (тройки, мастерские и т.д.) |
| **Формы заявок** (`.ajaxForm`) | 7 форм по странице |
| **Клиенты** (`.s26-hg8ap7v1d`) | Гарантия обращений |

#### 3. Формы
Все формы используют класс `.ajaxForm` и отправляют данные на `window.ajaxurl`. 
Для работы форм на вашем сервере замените `window.ajaxurl` на свой endpoint.

#### 4. Видео
Видео загружаются с `talentsy.ru`. Замените `data-src` атрибуты на свои URL.

#### 5. Виджет отзывов
Виджет `getreview` подключается через `data-widget-id="lkdsbysytBSxgsk9"`. 
Замените на свой ID или уберите секцию.

#### 6. CSS-классы
Все уникальные классы этого лендинга имеют суффикс `_hg8ap7v1d`.
При создании нового лендинга можно сохранить эти классы или создать свои.

### Шрифты

В шаблоне используются 3 семейства шрифтов, все файлы уже скачаны в папку `fonts/`:

| Шрифт | Файлы | Где используется |
|-------|-------|-----------------|
| **Atyp Display** | `AtypDisplay-Regular/Medium/Semibold.woff2` | Заголовки H1, H2 (класс `font-Atyp`) |
| **Atyp Text** | `AtypText-Regular/Medium/Semibold.woff2` | Подзаголовки и акцентный текст |
| **Manrope** | `Manrope-Regular/Medium/Bold.woff2` | Основной текст body, формы, кнопки |
| **DrukWideCyr** | `DrukWideCyr-Medium.woff` | Декоративный (редко, класс `druk`) |

Пути в CSS уже обновлены на локальные (`../fonts/`). Шрифты загружаются из:
- `css/landing.min.css` — Atyp Display, Atyp Text, Manrope
- `css/style-hg8ap7v1d.css` — Atyp Display (дополнительные начертания), Manrope
- `css/main.css` — DrukWideCyr

Структура папки:
```
fonts/
├── AtypDisplay-Regular.woff2   ← заголовки (нормальный)
├── AtypDisplay-Medium.woff2    ← заголовки (средний)
├── AtypDisplay-Semibold.woff2  ← заголовки (полужирный)
├── AtypText-Regular.woff2      ← подзаголовки
├── AtypText-Medium.woff2
├── AtypText-Semibold.woff2
├── Manrope-Regular.woff2       ← body текст
├── Manrope-Medium.woff2        ← кнопки, акценты
├── Manrope-Bold.woff2          ← жирный текст
├── DrukWideCyr-Medium.woff     ← декоративный
└── (+ .woff fallback для каждого)
```

### Важные заметки

- **Шрифты**: Все загружены локально, пути в CSS обновлены
- **Swiper**: Версия 12, подключается локально
- **jQuery**: Версия 3.6.4
- **Адаптивность**: Мобильная версия через классы `m-only-hg8ap7v1d` / `d-only-hg8ap7v1d`
- **Аккордеон**: Работает через `landing.min.js`
- **Калькулятор заработка**: Инициализируется через `noUiSlider` в `landing.min.js`
