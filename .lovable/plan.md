

## План: Редактирование офферов

Добавить кнопку редактирования на каждый оффер в списке (`OfferTypeDetail.tsx`), которая открывает диалог с предзаполненными данными.

### Изменения в `src/pages/OfferTypeDetail.tsx`

1. **Новое состояние** для редактирования: `editingOffer` (объект оффера или null), плюс состояния `editTitle`, `editDescription`, `editDocUrl`, `editSelectedTags`.

2. **Мутация `updateMutation`**:
   - `supabase.from("offers").update({ title, description, doc_url }).eq("id", offerId)`
   - Удалить старые `offer_tags` для этого оффера, вставить новые
   - Инвалидировать кеш

3. **Кнопка редактирования** (иконка `Pencil`) на каждой строке оффера — при клике заполняет поля из текущих данных и открывает диалог.

4. **Диалог редактирования** — аналогичен диалогу создания, но с заголовком «Редактировать оффер», предзаполненными полями и кнопкой «Сохранить».

Без изменений в БД — RLS-политика на UPDATE для `offers` уже есть (owner or admin).
