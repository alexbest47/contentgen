

## Проблема

На странице `OfferTypeDetail.tsx` кнопка «Создать» есть только для диагностик (строки 269-274). Для остальных типов офферов (мини-курс, вебинар и т.д.) кнопки создания нет — пропала.

## Решение

Добавить в `OfferTypeDetail.tsx` для не-диагностических типов:

1. **Состояние для диалога создания** — `createOpen`, `createTitle`, `createDescription`, `createDocUrl`, `createSelectedTags`
2. **Мутацию создания** — insert в таблицу `offers` + insert тегов в `offer_tags`
3. **Кнопку «Создать оффер»** в шапке (рядом с заголовком) для `!isDiagnosticType`
4. **Диалог с `OfferForm`** — форма уже есть в файле, нужно просто подключить её к новому диалогу

Компонент `OfferForm` уже определён в файле и используется для редактирования — его же используем для создания.
