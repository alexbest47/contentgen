

## Обновить Anthropic API key в проекте

### Проблема
Логи edge-функции показывают ошибку `"Your credit balance is too low"`, хотя баланс пополнен ($49.79). API key, сохранённый в проекте, мог потерять валидность или быть привязан к другому workspace.

### Действие
1. Перейди в [Anthropic Console → API keys](https://console.anthropic.com/settings/keys)
2. Создай новый API key (или скопируй существующий, если он точно из той же организации с балансом)
3. Я обновлю секрет `ANTHROPIC_API_KEY` в проекте через инструмент `add_secret`
4. После обновления — попробуй создать проект снова

Изменений в коде не требуется. Нужно только обновить значение секрета.
