

## Переименовать кнопку «Создать оффер» → «Добавить {тип}»

### Изменение в `src/pages/OfferTypeDetail.tsx` (строка ~320)

Заменить статический текст `Создать оффер` на динамический с использованием `getOfferTypeLabel(offerType)`:

```tsx
// Было:
Создать оффер

// Станет:
Добавить {getOfferTypeLabel(offerType ?? "")}
```

Например: «Добавить Мини-курс», «Добавить Вебинар», «Добавить Предсписок» и т.д.

Функция `getOfferTypeLabel` уже импортирована и возвращает нужные русские названия из `offerTypes.ts`. Кнопка по-прежнему не показывается для диагностик (`!isDiagnosticType`).
