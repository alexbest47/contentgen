

## Разделить управление программами и создание контента

### Суть
- «Создание контента» (`/programs`) — только выбор программы из списка (без кнопки «Создать»)
- «Платные программы» — новый пункт в «Подготовка контента», где можно создавать, редактировать и удалять программы

### Изменения

**1. `src/components/AppSidebar.tsx`**
- Добавить в `contentPrepNav` первым элементом: `{ title: "Платные программы", url: "/manage-programs", icon: GraduationCap }`

**2. `src/pages/Programs.tsx`**
- Убрать кнопку «Создать программу» и диалог создания
- Изменить заголовок: «Создание контента»
- Подзаголовок: «Выберите программу для создания контента»
- Если программ нет — показать сообщение «Нет программ. Создайте программу в разделе «Подготовка контента → Платные программы»»

**3. Новый файл `src/pages/ManagePrograms.tsx`**
- Копия текущего `Programs.tsx` с кнопкой «Создать программу» и диалогом создания
- Заголовок: «Платные программы», подзаголовок: «Управление образовательными программами»
- Клик по программе → `/manage-programs/:programId`

**4. Новый файл `src/pages/ManageProgramDetail.tsx`**
- Показывает информацию о программе (название, описание, Google Docs ссылки, теги)
- Кнопка редактирования с тем же диалогом что в текущем `ProgramDetail.tsx`
- Кнопка «Удалить программу»
- Кнопка «Назад» ведет на `/manage-programs`
- **Без** сетки офферов (контентные/продающие) — это остается в `ProgramDetail.tsx`

**5. `src/pages/ProgramDetail.tsx`**
- Убрать кнопку редактирования (карандаш) и диалог редактирования
- Оставить только отображение информации + сетку офферов

**6. `src/App.tsx`**
- Добавить маршруты:
  - `/manage-programs` → `<ProtectedRoute adminOnly><ManagePrograms /></ProtectedRoute>`
  - `/manage-programs/:programId` → `<ProtectedRoute adminOnly><ManageProgramDetail /></ProtectedRoute>`
