

## План: Автоматическое создание проекта при генерации лид-магнитов

### Суть изменения
Убрать ручное создание проектов на странице мини-курса. Вместо этого — кнопка "Сгенерировать лид-магниты", которая:
1. Создаёт новый проект автоматически
2. Генерирует название проекта через Lovable AI (Google Gemini Flash)
3. Запускает генерацию лид-магнитов
4. Переходит на страницу проекта

### Затрагиваемые файлы

#### 1. Новая edge-функция `supabase/functions/generate-project-name/index.ts`
- Принимает `course_title`, `program_title` 
- Вызывает Lovable AI Gateway (`google/gemini-3-flash-preview`) для генерации короткого названия проекта на русском
- Возвращает `{ name: "..." }`

#### 2. `supabase/config.toml`
- Добавить секцию `[functions.generate-project-name]` с `verify_jwt = false`

#### 3. `src/pages/CourseDetail.tsx`
- Убрать диалог создания проекта (Dialog, Input, Label)
- Заменить кнопку "Создать проект" на "Сгенерировать лид-магниты" (с иконкой Sparkles)
- Новая мутация:
  1. Вызвать `generate-project-name` → получить название
  2. Создать проект в `projects` с этим названием
  3. Вызвать `generate-lead-magnets` с `project_id`
  4. Навигация на страницу проекта (`/programs/.../projects/{id}`)
- Кнопка показывает прогресс: "Создание проекта..." → "Генерация лид-магнитов..."

#### 4. Без изменений
- `ProjectDetail.tsx` — остаётся как есть (показывает результаты)
- `generate-lead-magnets` — без изменений
- База данных — без изменений
