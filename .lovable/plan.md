

## План: Заменить карточки на список

Заменить grid-карточки на вертикальный список (bordered rows) на трёх страницах:

### 1. `src/pages/Programs.tsx` (строки 91-107)
- Убрать `grid gap-4 md:grid-cols-2 lg:grid-cols-3`
- Заменить на `<div className="border rounded-lg divide-y">` с элементами-строками
- Каждая строка: название, описание (line-clamp-1), дата, Google Doc индикатор, ChevronRight — всё в одну строку с `flex items-center justify-between`

### 2. `src/pages/ProgramDetail.tsx` (строки 168-181)
- Аналогично: убрать grid-карточки мини-курсов
- Список строк: название, описание (line-clamp-1), дата, ChevronRight

### 3. `src/pages/CourseDetail.tsx` (строки 116-132)
- Убрать grid-карточки проектов
- Список строк: название, Badge статуса, дата, ChevronRight

### Формат строки (общий паттерн)
```text
┌──────────────────────────────────────────────────────┐
│ Название                          Статус/мета    >   │
│ Описание (truncated)              Дата               │
├──────────────────────────────────────────────────────┤
│ ...                                                  │
└──────────────────────────────────────────────────────┘
```

Пустые состояния остаются без изменений. Диалоги и логика не меняются — только рендер списков.
