

## Обновить статические блоки под формат карточек письма

### Проблема
Сгенерированное ИИ письмо имеет цветной фон (из цветовой схемы) и белые карточки с закруглёнными углами поверх него. Статические блоки «Курсы на которые идёт набор» и «Подборка бесплатных курсов» создают свой собственный белый контейнер, но без внешнего цветного фона — из-за этого они выглядят чужеродно.

### Решение
Обернуть HTML обоих блоков во внешнюю таблицу с фоновым цветом из цветовой схемы (`preview_colors[1]`), а внутри — белая карточка с `border-radius:12px`. Это воспроизводит ту же структуру «фон → карточка», которую использует ИИ-генерация.

### Файлы и изменения

**1. `src/components/email-builder/PaidProgramsCollectionSettings.tsx`** — обновить `buildHtml()`:
- Обернуть текущий HTML во внешнюю таблицу: `<table width="100%" style="background-color:${bgColor};"><tr><td style="padding:16px 0;">` ... `</td></tr></table>`
- `bgColor` = `schemeColors?.[1] || "#F0EDF7"` (вторичный цвет схемы)
- Внутренняя карточка остаётся: `background:#FFFFFF;border-radius:12px;padding:24px 32px`

**2. `src/components/email-builder/FreeCoursesGridSettings.tsx`** — аналогичное обновление `buildHtml()`:
- Та же обёртка с `bgColor` из `schemeColors?.[1]`
- Внутренняя карточка без изменений

Результат: оба блока будут визуально продолжать цветной фон письма с белыми карточками поверх, как это делают сгенерированные ИИ блоки.
