

## Перенос создания офферов в «Подготовка офферов» (по аналогии с диагностиками)

### Суть изменения

Сейчас офферы создаются внутри программы (Программа → Тип оффера → «Добавить»). Нужно перенести создание/редактирование/архивирование офферов на отдельные страницы в разделе «Подготовка офферов», а в программе оставить только выбор из уже существующих офферов — по аналогии с тем, как работают диагностики.

### Файлы и изменения

**1. `src/components/AppSidebar.tsx`** — расширить `offerPrepNav`

Добавить 7 пунктов (иконки из `offerTypes.ts`):
- Мини-курс → `/offers/mini_course`
- Вебинар → `/offers/webinar`
- Скачай PDF → `/offers/download_pdf`
- Предсписок → `/offers/pre_list`
- Старт нового потока → `/offers/new_stream`
- Освободилось место → `/offers/spot_available`
- Промокод → `/offers/discount`

Диагностики остаются как есть (`/diagnostics`).

**2. Новая страница `src/pages/OfferTypeManagement.tsx`**

Универсальная страница управления офферами одного типа. Получает тип из URL-параметра `:offerType`. Функционал:
- Список всех офферов данного типа (из всех программ, не архивных), с указанием программы
- Кнопка «Добавить» → диалог создания (выбор программы, название, описание, изображение для контентных типов, Google Doc, теги)
- Редактирование и архивирование существующих офферов
- Переход к деталям оффера при клике

По сути — перенос CRUD-логики из текущего `OfferTypeDetail.tsx` в отдельную страницу с добавлением выбора программы при создании.

**3. `src/App.tsx`** — новые маршруты

Добавить:
```
/offers/:offerType → OfferTypeManagement (adminOnly)
```

**4. `src/pages/OfferTypeDetail.tsx`** — убрать создание/редактирование

Для не-диагностических типов:
- Убрать кнопку «Добавить», диалоги создания и редактирования
- Оставить только список офферов для выбора (клик → переход к `OfferDetail`)
- Убрать кнопки редактирования/архивирования из списка
- Добавить пустое состояние с текстом «Нет офферов. Создайте их в разделе "Подготовка офферов"»

**5. `src/pages/ProgramDetail.tsx`** — без изменений

Карточки типов офферов остаются, клик по ним ведёт к `OfferTypeDetail` (теперь только для выбора).

### Итого
- 1 новый файл (`OfferTypeManagement.tsx`)
- 3 файла изменены (`AppSidebar.tsx`, `App.tsx`, `OfferTypeDetail.tsx`)
