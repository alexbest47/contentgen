

## Исправление маппинга в import-prompts-txt

Секция `"Генерация справочных материалов"` не распознаётся, потому что в маппинге записано `"Генерация справочного материала"`.

### Изменение

**`supabase/functions/import-prompts-txt/index.ts`**: добавить в `mapSection` альтернативное написание:

```
"Генерация справочных материалов": { content_type: "reference_material", channel: null },
```

Одна строка в маппинге — и все промпты будут распознаны.
