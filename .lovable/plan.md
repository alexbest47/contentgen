

## Показ выбранного кейса на странице проекта

После того как пользователь выбрал кейс, страница проекта (`ProjectDetail.tsx`) должна отображать информацию о выбранном кейсе.

### Что нужно сделать

**`src/pages/ProjectDetail.tsx`:**

1. Добавить `useQuery` для загрузки выбранного кейса из `case_classifications` по `project.selected_case_id` (когда `selected_case_id` существует).

2. После заголовка страницы и перед секцией углов подачи — показать карточку «Выбранный кейс» с:
   - Имя файла (`file_name`)
   - Ссылка на видеофайл (`source_url`) — если есть, открывается в новой вкладке с иконкой ExternalLink
   - Краткое описание (`classification_json.summary`)
   - Имя студента, тип видео, продукты — компактно в бейджах
   - Кнопка «Посмотреть JSON» для просмотра полной классификации

3. Карточка отображается для `testimonial_content` проектов, у которых `selected_case_id` не null (то есть на всех этапах после выбора кейса: `leads_ready`, `lead_selected`, `completed`).
