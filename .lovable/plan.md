

## Добавить редактирование названия и описания диагностик

На странице `/diagnostics` добавить диалог редактирования, открываемый по кнопке (иконка Pencil) в колонке действий.

### Изменения в `src/pages/Diagnostics.tsx`

1. **Состояние**: добавить `editingDiag` (`{ id, name, description } | null`) для хранения редактируемой диагностики
2. **Мутация**: `updateMutation` — обновляет `name` и `description` в таблице `diagnostics`, инвалидирует кеш
3. **Кнопка**: добавить иконку `Pencil` в колонку действий (рядом с "Открыть" и корзиной), открывающую диалог с предзаполненными полями
4. **Диалог**: `Dialog` с двумя полями — `Input` (название) и `Textarea` (описание), кнопка «Сохранить»

Никаких изменений БД не требуется — поля `name` и `description` уже есть в таблице `diagnostics`, RLS разрешает update владельцу и админу.
