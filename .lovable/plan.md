

## Изменение порядка шагов в Wizard

Текущий порядок: 1. Тема → 2. Аудитория → 3. Шаблон → 4. Настройки

Новый порядок: **1. Шаблон → 2. Тема → 3. Аудитория → 4. Настройки**

### Изменения в `src/components/email-builder/CreateLetterWizard.tsx`

**1. Заголовки шагов** (строки 300-308): обновить `stepTitle`:
- Шаг 1 → «Как построить письмо?» (шаблон)
- Шаг 2 → «О чём это письмо?» (тема)
- Шаг 3 → «Для кого это письмо?» (аудитория)
- Шаг 4 → «Настройки» (без изменений)

**2. Условия загрузки данных** (`enabled`):
- `templates` — загружать с шага 1 (`step >= 1`)
- `topicRows` — уже загружается всегда при `open`, без изменений
- `audienceVars` — загружать с шага 3 (`step >= 3`)

**3. Рендер шагов** (строки 324-513): переставить блоки:
- `step === 1` → рендерить шаблон (бывший step 3)
- `step === 2` → рендерить тему (бывший step 1)
- `step === 3` → рендерить аудиторию (бывший step 2)
- `step === 4` → настройки (без изменений)

**4. Кнопки навигации** (строки 516-542):
- Шаг 1: «Далее» по `canNext3` (выбран шаблон) → `setStep(2)`
- Шаг 2: «Далее» по `canNext1` (есть тема) → `setStep(3)`. Учесть `themeOnlyMode`.
- Шаг 3: «Далее» по `canNext2` (выбрана аудитория) → `setStep(4)`
- Шаг 4: «Создать письмо» (без изменений)

**5. `themeOnlyMode`**: этот режим используется только для выбора темы — он теперь на шаге 2. Обновить проверки соответственно.

Объём: 1 файл, ~60 строк перестановок.
