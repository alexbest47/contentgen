

## Добавить кнопку «Скачать» для баннеров

### Что делаем
Добавить пункт «Скачать» в выпадающее меню карточки баннера. По клику — скачивать оригинальное полноразмерное изображение по `image_url`.

### Изменения

**`src/pages/BannerLibrary.tsx`**

1. Импортировать иконку `Download` из `lucide-react`.
2. В `BannerGrid` добавить `<DropdownMenuItem>` «Скачать» с иконкой `Download`.
3. Реализовать скачивание через `fetch` → `blob` → `URL.createObjectURL` → программный клик по `<a download>`. Это нужно потому, что прямая ссылка на Supabase Storage откроется в браузере, а не скачается. Имя файла формируется из `b.title` + расширение из URL.

```text
Меню карточки:
  ├── Перегенерировать (если generated)
  ├── Скачать              ← новый пункт
  ├── Редактировать метаданные
  └── Удалить
```
