

## Добавить алиасы для промптов «Контент-отзыв»

### Проблема
Заголовки в TXT-файле не совпадают с именами в базе:

| В файле | В базе |
|---|---|
| Генерация углов подачи кейса | Генерация углов отзыва |
| Кейс / отзыв: Instagram | Контент-отзыв: Instagram |
| Кейс / отзыв: Telegram | Контент-отзыв: Telegram |
| Кейс / отзыв: VK | Контент-отзыв: VK |
| Кейс / отзыв: Email | Контент-отзыв: Email |

### Решение

**`supabase/functions/import-prompts-txt/index.ts`**: добавить 5 записей в `NAME_ALIASES`:

```typescript
"Генерация углов подачи кейса": "Генерация углов отзыва",
"Кейс / отзыв: Instagram": "Контент-отзыв: Instagram",
"Кейс / отзыв: Telegram": "Контент-отзыв: Telegram",
"Кейс / отзыв: VK": "Контент-отзыв: VK",
"Кейс / отзыв: Email": "Контент-отзыв: Email",
```

Одно изменение в edge-функции, редеплой.
