

## Добавить карточку «Экспертный контент» в справочник переменных

Добавить новую категорию в массив `categories` в файле `src/pages/PromptVariables.tsx`:

```typescript
{
  title: "Экспертный контент",
  variables: [
    { 
      name: "{{expert_post_topic}}", 
      description: "Полный контекст выбранной темы экспертного поста (категория, угол подачи, крючок, переход к офферу)", 
      source: "Сгенерированные данные (lead_magnets)" 
    },
  ],
}
```

Карточка появится после «Справочные материалы» и перед «Диагностики» — один файл, одна вставка.
