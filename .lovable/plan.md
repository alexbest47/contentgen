

## Перенести кнопку «Сгенерировать лид-магниты» в строку каждого оффера на странице списка

Кнопка «Сгенерировать лид-магниты» находится на странице `OfferDetail.tsx` (страница проектов конкретного оффера). Она генерирует проект с лид-магнитами для этого оффера.

### Что нужно
Перенести кнопку из `OfferDetail.tsx` в `OfferTypeDetail.tsx` — в строку каждого оффера в списке, для всех не-диагностических типов. Так пользователь сможет запускать генерацию прямо из списка офферов, не заходя внутрь.

### Изменения

- Перенести `generateMutation` из OfferDetail (создание проекта + генерация лид-магнитов)

- После генерации — навигация на страницу проекта

**2. `src/pages/OfferDetail.tsx`** — убрать кнопку:
- Удалить блок с кнопкой «Сгенерировать лид-магниты» (строки 144-152)

- Упростить импорты

### Визуально
