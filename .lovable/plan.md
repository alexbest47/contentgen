

## Фильтр по тегам на странице «Платные программы»

Добавить ряд кликабельных бейджей-тегов над списком программ. При клике на тег — список фильтруется, показывая только программы с этим тегом. Можно выбрать несколько тегов (AND-логика или OR — выбираем OR, чтобы показывать программы с любым из выбранных тегов). Повторный клик снимает фильтр.

### Что увидит пользователь

Между заголовком и списком программ появится горизонтальный ряд бейджей с тегами (из уже загруженного `allTags`). Выбранные теги подсвечиваются (variant="default"), невыбранные — outline. Список программ фильтруется на клиенте.

### Технические детали

**Файл: `src/pages/ManagePrograms.tsx`**

1. Добавить состояние `filterTagIds: string[]`.
2. Между заголовком и списком — блок с бейджами тегов (из `allTags`), клик переключает тег в `filterTagIds`.
3. Вычислить `filteredPrograms`: если `filterTagIds` пуст — все программы; иначе — только те, у которых хотя бы один тег из `filterTagIds` есть в `program_tags`.
4. Использовать `filteredPrograms` вместо `programs` в рендере списка.
