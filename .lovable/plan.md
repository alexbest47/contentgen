

## Задача

123 из 206 кейсов имеют пустое `student_name`. Нужно извлечь имя из `file_name` и записать в `classification_json`.

## Анализ паттернов имён файлов

Типичные форматы:
- `25. Психолог-консультант _ Наталья Иванова.mp4` → Наталья Иванова
- `14. Психолог-консультант_Жанна Абрамова.mp4` → Жанна Абрамова
- `Виктория Малоземова.mp4` → Виктория Малоземова
- `4. КСО Валентина Барболина.mp4` → Валентина Барболина
- `Дарья.MOV` → Дарья
- `ПК. Ольга Киркина.MOV` → Ольга Киркина
- `IMG_0132.MP4` → невозможно извлечь
- `20230216_172118_1.mp4` → невозможно извлечь

## Подход

Напишу Python-скрипт, который:

1. Получит все 123 записи с пустым `student_name`
2. Для каждого `file_name` применит цепочку правил:
   - Убрать расширение (.mp4, .mov, .mkv)
   - Убрать нумерацию в начале (`25.`, `02.`)
   - Убрать известные категории (Психолог-консультант, Нутри, Нутр, КСО, Интервью выпускника, Семейная Пси, Интегративный нутрициолог, ПК)
   - Убрать разделители (`_`, `-`, `.`, `,`)
   - Оставшийся текст — кириллические слова с заглавной буквы → это имя
   - Если ничего не нашлось — пропустить
3. Выведет таблицу `file_name → extracted_name` для проверки
4. После проверки обновит `classification_json` через SQL UPDATE

Скрипт запущу через `lov-exec`, обновления — через insert tool. Сначала покажу результат парсинга, чтобы вы могли проверить перед записью в БД.
