

## Восстановить кнопку «Сгенерировать лид-магниты» на странице OfferDetail

### Проблема
При переносе кнопки генерации на страницу списка офферов (`OfferTypeDetail`) она была полностью удалена со страницы конкретного оффера (`OfferDetail`). Пользователь хочет видеть кнопку и там тоже.

### Решение
Вернуть кнопку «Сгенерировать лид-магниты» в `src/pages/OfferDetail.tsx`:

1. Добавить состояния `generatingProject` и `progressText`
2. Восстановить `generateMutation` (аналогичную той, что в OfferTypeDetail): вызов `generate-project-name`, создание проекта, вызов `generate-lead-magnets`, навигация на страницу проекта
3. Добавить кнопку с иконкой `Sparkles` в шапку страницы (рядом с заголовком оффера), показывать для всех типов кроме `diagnostic`
4. Импортировать `Sparkles` из lucide-react

Кнопка будет дублировать ту, что на странице списка — пользователь сможет запустить генерацию с обоих экранов.
